# msc_real_estate
## Pet-проект по GeoData Analytics
О проекте

Проект посвящён анализу факторов, влияющих на стоимость жилой недвижимости, и построению моделей для прогнозирования цены за квадратный метр.

В работе используются методы пространственного анализа, пространственной эконометрики и машинного обучения, что позволяет учитывать не только характеристики квартиры, но и пространственную структуру рынка недвижимости.

## Основные этапы

1. Предобработка данных

- очистка данных

- расчет расстояния до метро

- расчет расстояния до центра города

- построение пространственного лага цен

2. Пространственный эконометрический анализ

- тест на пространственную автокорреляцию (Moran's I)

- OLS модель

- Spatial Lag и Spatial Error модели

3. Машинное обучение

- базовая модель на числовых признаках

- улучшенная модель с учетом района (categorical feature)

- интерпретация модели с помощью SHAP

## Основные результаты

Анализ показывает, что цены на недвижимость обладают выраженной пространственной структурой.

Ключевые факторы, влияющие на стоимость жилья:

- расстояние до центра города

- транспортная доступность (близость к метро)

- локальный ценовой уровень соседних объектов

- район расположения недвижимости

Использование пространственных признаков и районных характеристик позволяет существенно улучшить качество моделей прогнозирования.

## Используемые технологии

Python, pandas, geopandas, scikit-learn, statsmodels, PySAL, SHAP, osmnx, plotly.
